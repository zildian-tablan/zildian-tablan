## 📌 Academic Projects

### 📱 [BulSU Space](https://github.com/zildian-tablan/BulSU-Space)
BulSU Space is a university communication platform designed to provide students, faculty, and alumni with a centralized space for academic collaboration and campus engagement. The platform allows users to create and interact with community posts, receive university announcements, exchange private messages, and participate in discussions. To improve user engagement, the system delivers real-time push notifications for new announcements and community activities. My primary contribution was implementing the push notification system, as well as assisting in debugging, testing, and enhancing the overall user experience.

### 🛒 [Core Gear Tech](https://github.com/zildian-tablan/CoreGearTech)
Core Gear Tech is an e-commerce platform developed to simplify the online shopping experience for customers while providing administrators with efficient management tools. Users can browse products by category, search for specific items, add products to their shopping cart, manage their orders, and securely access their accounts. Administrators can manage product listings, monitor customer orders, and oversee inventory through a dedicated dashboard. The project focuses on creating a seamless shopping experience while maintaining an organized system for store management.

### 🎮 [RuneBound](https://github.com/zildian-tablan/Rune-Bound)
RuneBound is a 2D action-platformer game where players explore different levels filled with obstacles, enemies, and environmental challenges. The game features smooth character movement, combat mechanics, collectible items, checkpoints, and progressive level difficulty to keep gameplay engaging. The project emphasizes responsive controls, level design, and an enjoyable player experience while demonstrating core game development principles.

### 🧠 [Elevating IQ](https://github.com/zildian-tablan/Elevating-IQ)
Elevating IQ is an educational quiz application designed to make learning interactive and enjoyable. It presents users with questions from different categories, tracks their scores, and provides immediate feedback after each answer. The application encourages continuous learning by allowing users to challenge themselves, monitor their progress, and improve their knowledge through repeated practice in a simple and engaging environment.

### 🚢 [Titan Cruise](https://github.com/zildian-tablan/TITAN-Cruise)
Titan Cruise is a reservation management system created to streamline the cruise booking process for both customers and administrators. The system enables users to browse available cruise packages, make reservations, manage booking details, and maintain customer records efficiently. Administrators can organize reservations, update trip information, and monitor customer bookings, resulting in a more organized and user-friendly reservation process.
